## ADDED Requirements
### Requirement: Mobile-friendly admin tables
The system SHALL keep admin data tables usable on mobile by enabling horizontal scrolling.

#### Scenario: View admin tables on mobile
- **WHEN** the user views admin tables on a mobile viewport
- **THEN** the tables can be scrolled horizontally without breaking layout

### Requirement: Admin styling aligned with main theme on mobile
The system SHALL align admin page styling with the main app theme on mobile viewports.

#### Scenario: Open admin on mobile
- **WHEN** the admin page loads on a mobile viewport
- **THEN** typography and surface styling match the main app visual system
